# Hassan Mohammed Said - Portfolio

This is a **personal portfolio** of Hassan Mohammed Said, an ICT professional and aspiring IT manager.  

It is a **single-page website** built using **HTML and CSS**, showcasing:  

- About Me  
- Skills  
- Education  
- Contact Form  

## Features

- Colorful sections with hover effects  
- Skills listed using HTML lists  
- Education displayed in a table  
- Contact form with radio buttons and submit button  

## How to View

1. Clone or download this repository.  
2. Open `index.html` in any web browser.  

## Technologies Used

- HTML  
- CSS
